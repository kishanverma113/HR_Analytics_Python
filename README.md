# 📊 HR Analytics using Python

> **An end-to-end HR Analytics project built with Python, Pandas, NumPy, Matplotlib, and Seaborn to analyze employee data, perform Exploratory Data Analysis (EDA), and generate actionable business insights through statistical analysis and interactive visualizations.**

---

## 📌 Project Overview

Human Resource (HR) Analytics helps organizations understand employee behavior, identify workforce trends, and make data-driven decisions.

In this project, a real-world HR dataset is analyzed using Python. The project follows a complete data analysis workflow, including data loading, cleaning, preprocessing, statistical analysis, visualization, and business insight generation.

This repository is designed to demonstrate practical Data Analytics skills that are commonly used in real business environments.

---

# 🎯 Project Objectives

The main objectives of this project are to:

* Understand the HR dataset structure.
* Perform data cleaning and preprocessing.
* Analyze employee demographics.
* Study employee attrition patterns.
* Explore salary distribution.
* Compare department performance.
* Perform statistical analysis.
* Create professional visualizations.
* Generate meaningful business insights.
